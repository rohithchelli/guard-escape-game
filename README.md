**Guard Escape**

A browser-based stealth escape game built using HTML, CSS, and JavaScript. The player must move through a guarded area, avoid enemy guards, collect keys, unlock doors, and reach the exit without being caught.

**Tech Used:** HTML, CSS, JavaScript
**Project Type:** Browser Game
**Status:** Playable Prototype

**Key Features:**

* Player movement using keyboard controls
* Guard patrol system
* Collision detection
* Key collection and locked doors
* Win and game-over conditions
* Simple level-based gameplay
* Clean game UI with score/status display
